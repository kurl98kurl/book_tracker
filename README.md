# Трекер прочитанных книг (Book Tracker)

Консольное Python-приложение для удобного учета и ведения списка прочитанных книг с модульной архитектурой.

## Особенности проекта
* **Модульная структура**: Логика разделена на отдельные файлы (`models.py`, `storage.py`, `stats.py`, `main.py`).
* **Хранение данных**: Информация сохраняется в локальный файл `books.json`.
* **Защита от дубликатов**: Программа не позволит добавить одну и ту же книгу одного автора дважды.
* **Статистика**: Автоматический подсчет общего количества книг, среднего балла и количества книг по каждому автору.

## Как запустить проект

1. Убедитесь, что у вас установлен **Python 3**.
2. Скачайте файлы проекта в одну папку.
3. Откройте терминал/консоль в этой папке и запустите команду:
   ```bash
   python main.py
   ```

## Архитектура проекта
* `main.py` — Главный интерфейс программы и текстовое меню.
* `models.py` — Описание сущности (класса) Книги.
* `storage.py` — Функции сохранения, загрузки данных и валидации дубликатов.
* `stats.py` — Расчет аналитики и выгрузка статистики.
* `.gitignore` — Исключение временных файлов Python и базы данных JSON из репозитория.
